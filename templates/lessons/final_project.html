Description

For the Final Project, you will design and code a program using the tools weâ€™ve learned about in this class. Working in a group of 3-4 students, you must think of a problem that you could solve with a computer program. The problem should be easy enough to solve with the tools you have, but hard enough that you will have to use many tools. Here are a couple of examples:

Use random numbers to make a dice or card game. Make a program that solves a math problem for you (The Ackermann Function is a good one!)

Grading

Grading will be in two parts:

  • Flowchart of your program (30 points)
  • Code for your program (70 points)

For the Code part of your program, you will be graded on how many tools you use, and on how well you use them. You DO NOT need to use every tool. Only enough to get you 70 points. Make sure you try to use the Right Tool For The Job.

Basic Tools

Use enough tools to get 70 points. You DO NOT need to use all of them

  • Input/Output (20 points)
  • 2 if-statements (10 points)
  • 2 Logical Operators (AND - &&, OR - ||, NOT - !) (10 points)
  • A switch-case statement (10 points)
  • A While Loop or a For Loop (10 points)
  • A Function (10 points)
  • An Array that is not a String (10 points)
  • Random Numbers (20 points)
  • File Input/Output (20 points)
  • Recursion (20 points)

Advanced Tools:

(see me if you want to do any of these)

  • Classes/Structs with Pointers (70 points)
  • Fetch Data from the Internet (with cURLlib) (70 points)
  • Use UNICODE and wide characters to read/write Thai letters (70 points)
  • Use NCURSES to make a program with a screen. (70 points)
